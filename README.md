# 408-FARMERS V2 Website

Static site for Cloudflare Pages / GitHub Pages.

## Files

- `index.html` — landing page with pre-filled text buttons
- `styles.css` — styling
- `contact.vcf` — downloadable contact card
- `assets/dylan-headshot.png` — headshot image

## How to deploy with GitHub + Cloudflare Pages

1. Create a new GitHub repo.
2. Upload all files/folders from this package.
3. In Cloudflare Pages, create a new project connected to the repo.
4. Framework preset: **None**
5. Build command: leave blank
6. Output directory: `/`
7. Deploy.
8. Point `408farmers.com` to the Cloudflare Pages project.
